# NetLogo Models
Here you will find the four NetLogo models used on the "The role of incentives in electoral dynamics: An agent-based and replicator equation approach" project

-The first one, which is called "Prototyp 14AP ALL PARTIES equity" contains the model where the three parties have the exact number of followers
-The second one, called "Prototyp 14AP ALL PARTIES" states a model of the electoral process in Mexico in 2024
-While the third archive, which name is "Prototyp 14AP ALL PARTIES A_B" is a modified version of the last one, where the majority of the population belong to the highest socioeconimic level 
-Finally, the fourth archive, which is called "Prototyp 14AP ALL PARTIES E" states the electoral process in Mexico in 2024 where the majority of the population belong to the lowest socioeconomic level
